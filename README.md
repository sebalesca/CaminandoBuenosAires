### Node.Js - /server

### Como empezar?

* Instala todas las librerias. Utilizando el comando

```bash
$ npm install
```
 en la carpeta principal del repositorio.  


* Inicia el servidor nodejs utilizando el comando

```bash
$  npm start
```
